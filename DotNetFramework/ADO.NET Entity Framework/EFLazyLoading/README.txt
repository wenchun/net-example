This is a sample implementation of lazy loading extension for Entity Framework.
See http://code.msdn.microsoft.com/EFLazyLoading for more information.

Directory layout:

    Help - API documentation (generated using Sandcastle)
    Database - NorthwindEF database
    Source - source code for Microsoft.Data.EFLazyLoading.dll and EFLazyClassGen.exe
    Tests - various tests that exercise the lazy loading functionality
